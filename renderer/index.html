⚡ LifeServer

LifeServer is a very simple live server application that can launch a development local server with live reload feature for static & dynamic pages. Works on macOS, Linux, and Windows.

  1. Enter the port number you want the server to run on.
  2. Enter the host to bind to (e.g. localhost, 0.0.0.0, 127.0.0.1).
  3. Enter the directory path of your project (or leave . for current folder).
  4. Click "Start Server" and your site will open in the browser with live reload.
